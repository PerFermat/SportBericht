📋 EMPFOHLENE VORGEHENSWEISE
─────────────────────────────────────────────────────────────────────────────────────────
  1. Feature-Branch erstellen: git checkout -b refactor/german-naming
  2. Jede Klasse einzeln refaktorieren
  3. Nach jeder Klasse: mvn clean compile
  4. Nach Klasse-Gruppe: Alle Tests ausführen
  5. Regelmäßige Commits: git commit -m "refactor: rename [ClassName] to German"
  6. Pull Request erstellen zum Mergen in main-Branch
